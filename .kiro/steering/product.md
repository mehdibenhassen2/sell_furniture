# Product Overview

Furniture sale/moving sale application for listing and browsing items for sale. Users can browse items by category, search listings, and authenticated users can add new items. Includes delivery information and visitor tracking.

## Key Features

- Browse furniture items with image galleries
- Category-based filtering (Bed, Dresser, Technology, Decoration, etc.)
- Search functionality
- User authentication (register/login)
- Protected routes for adding new items
- Delivery information page
- Visitor analytics tracking
