# Tech Stack

## Frontend

- **Angular 17.2** (standalone components)
- **TypeScript 5.3** with strict mode enabled
- **SCSS** for styling
- **RxJS 7.8** for reactive programming
- **Angular Router** with lazy loading
- **Angular Signals** for state management
- **FontAwesome** for icons

## Backend

- **Node.js** (>=18.0.0)
- **Express** server
- **MongoDB** with Mongoose
- **JWT** authentication with bcrypt
- **Angular SSR** (Server-Side Rendering)

## Build System

- **Angular CLI** for build and dev tooling
- **Vite** (via Angular 17 builder)
- **Karma + Jasmine** for testing

## Common Commands

```bash
# Development
npm start              # Start dev server (http://localhost:4200)
ng serve              # Alternative dev server command

# Build
npm run build         # Production build
ng build              # Development build

# Testing
npm test              # Run unit tests
ng test               # Run tests with Karma

# Code Quality
npm run lint          # Lint code
ng lint               # Alternative lint command

# Code Generation
ng generate component <name>   # Generate component
ng generate service <name>     # Generate service
ng generate pipe <name>        # Generate pipe
ng generate guard <name>       # Generate guard
```
